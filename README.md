PhotoChoice
=====================

## 描述

打造超级无敌牛逼的图片选择，多选图片，再也不需要系统自带图库啦，点击 gridview 的 item 可以选择拍照选择还是从相册选择。通过广播发给对应的 activity，好用好用再好用。
